hi, 請看我本次增加了什麼。我用這個做一個範例來展示 git。 prettier 有先找過網路的資料嗎？ 方法1: 請先在自己的專案資料夾下安裝 prettier `npm install --save-dev prettier` 方法2: 使用 cursor or VScode ，安裝裡面的套件 1. 安裝 VS Code 套件：Prettier - Code formatter 2. 在 .vscode/settings.json 中加入： { "editor.formatOnSave": true, "editor.defaultFormatter": "esbenp.prettier-vscode" }
